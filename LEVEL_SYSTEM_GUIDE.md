## 📊 How It Works

### XP (Experience Points)

Every activity you log earns you XP:

**Base Formula:**
```
1 minute of activity = 1 base XP
```

**Category Multipliers:**
- 📚 **Study**: 1.5x XP (Best for leveling!)
- 💪 **Fitness**: 1.3x XP 
- 🧘 **Mindfulness**: 1.2x XP

**Examples:**
```
90 min Study Session
= 90 × 1.5 = 135 XP ✨

60 min Gym Workout  
= 60 × 1.3 = 78 XP 💪

15 min Meditation
= 15 × 1.2 = 18 XP 🧘
```

### Level Progression

Levels use **exponential growth** - each level requires more XP than the last:

```
Level 1 → 2:   100 XP
Level 2 → 3:   283 XP
Level 5 → 6:   1,118 XP
Level 10 → 11: 3,162 XP
Level 50 → 51: 35,355 XP
Level 99 → 100: 99,000 XP
```

**Total XP to reach Level 100:** ~493,000 XP

## 💡 Leveling Strategy

### Fast Leveling Tips

**1. Study Sessions (Best XP)**
```
Daily Goal: 150 min study
= 150 × 1.5 = 225 XP/day
= 1,575 XP/week
```

**2. Combine Categories**
```
90 min study + 60 min fitness + 15 min meditation
= 135 + 78 + 18 = 231 XP in one day!
```

**3. Consistency Beats Intensity**
```
Option A: 200 min on one day = 300 XP
Option B: 50 min × 7 days = 525 XP/week ✅ Better!
```

**A:** No, this is your personal journey! All data is local.

### Q: What happens after Level 100?
